About Me

Updated: January 20, 2024

Hi, I’m Sam Feldstein. I used to want to be an author, but lately I’m more interested in web design. I’m fond of blogs, books, movies, and photography.

The best way to keep up with my work is to subscribe to my RSS feed. If you don’t use RSS, I also have a mailing list.

Things I Use

I don’t write fiction so much anymore, but when I did I used Scrivener. For distraction-free word-dumping, I use Enso. For writing everything else, taking notes, and managing files, I use Obsidian with the Minimal theme and the following community plugins:

  • Hider
  • LanguageTool Integration
  • Minimal Theme Settings
  • Omnivore
  • Readwise Official
  • Typewriter Mode

My go-to dictionaries are:

  • Dictionary.com
  • The Free Dictionary
  • Merriam-Webster
  • The New Oxford American Dictionary (macOS)
  • The Oxford Dictionary of English (British English, macOS)
  • Webster’s 1913 Dictionary (macOS)
  • Roget’s Thesaurus
  • Thesaurus.com
  • Online Etymology Dictionary

I track RSS feeds with NewsBlur. I used to read and annotate articles with Readwise but recently switched to Omnivore. Omnivore is open source, feels lighter, and so far doesn’t include any annoying features you can’t turn off (a la the blue bar to the left of paragraphs in Readwise). It doesn’t support video transcription, so I’ll keep using Readwise for that and also for its daily review feature. I store my notes and highlights in Obsidian using the plugins mentioned above.

For memorizing facts, quotes, and other information, I use spaced repetition via Anki.

For coding, I use VS Code.

I built this website with Hugo and a custom theme.

Writing Checklist

For context, I built my website with Hugo and custom theme.

  1. Draft in Obsidian
  2. Paste into Hemingway
  3. Run LanguageTool extension
  4. Run Grammarly extension
  5. Paste into VS Code
  6. Check Problems tab
  7. Check for broken links
  8. Submit for publication

Publish on Blog

  1. Add description
  2. Add deck
  3. Add tags
  4. Change date
  5. Run hugo server -D, make sure everything looks okay
  6. Change draft status
  7. Update file name
  8. Commit and push
  9. Send email
  10. Promote
